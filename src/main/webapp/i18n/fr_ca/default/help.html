JUICY LOOT

Apportez-moi ce butin juteux! Passez un délicieux moment en jumelant des fruits et en remplissant les commandes de jus pour obtenir des scores élevés. Amassez des verres de jus durant le jeu principal pour gagner des lots sucrés durant le jeu boni. Vous pourriez même gagner un succulent lot instantané!

Comment miser

Coût du billet

Affiche le coût actuel du billet.

Appuyez sur la flèche gauche (–) pour diminuer le coût du billet.

Appuyez sur la flèche droite (+) pour augmenter le coût du billet.

Appuyez sur ACHETER pour débiter le prix du billet de votre solde et commencer la partie.

Comment jouer

Vous commencez la partie avec 25 déplacements.

Le but du jeu est de remplir les commandes de jus en interchangeant les fruits pour les jumeler. Un mouvement correspond à un échange. Lorsque trois fruits identiques ou plus apparaissent dans une rangée ou une colonne, ils disparaissent de l’écran.

Les commandes de jus de chaque fruit sont affichées à l’écran. Les fruits qui ont été jumelés et qui ont disparu de l’écran sont amassés dans les commandes de jus correspondantes. Chaque fois que 8, 25 et 40 fruits d’un type de fruit sont amassés, un verre de jus est accordé. Un type de fruit peut accorder un maximum de 3 verres de jus. Aucun verre de jus supplémentaire n’est accordé si plus de 40 fruits d’un type de fruit ont été amassés.

Un minimum de 5 verres de jus et un maximum de 18 verres de jus sont accordés par partie. Chaque verre de jus accorde une sélection supplémentaire durant le jeu boni.

Une fois les 25 sélections effectuées, le jeu boni est déclenché. Trouvez les verres de jus pleins durant le jeu boni et trouvez trois lots identiques pour gagner. Si un lot mauve instantané est révélé, le lot affiché est accordé instantanément.

Échange automatique

Appuyez sur ÉCHANGE AUTOMATIQUE pour que le jeu principal se joue automatiquement jusqu’à la fin. Le jeu continue jusqu’à ce que tous les déplacements soient complétés. Appuyez sur ARRÊTER à tout moment pour continuer le jeu manuellement.

Tout afficher

Appuyez sur TOUT AFFICHER pour que le jeu boni se joue automatiquement jusqu’à la fin. Le jeu se poursuit jusqu’à ce que toutes les sélections aient été révélées et que tous les lots aient été accordés.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les lots instantanés sont ajoutés aux gains totaux.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
